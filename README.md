# DSA

A clean, topic-wise repository structure for practicing data structures and algorithms.

## Repo structure

- `algorithms/` — algorithm practice by category
- `data-structures/` — data structure problems and implementations
- `cheatsheet.md` — last-minute revision summary
- `README.md` — this repository usage guide
